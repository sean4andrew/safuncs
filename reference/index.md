# Package index

## Survival Analysis

Functions for generating, plotting, and simulating survival data

- [`Surv_Gen()`](https://sean4andrew.github.io/safuncs/reference/Surv_Gen.md)
  : Generate Survivor Data
- [`Surv_Plots()`](https://sean4andrew.github.io/safuncs/reference/Surv_Plots.md)
  : Generate Survival Plots
- [`Surv_Power()`](https://sean4andrew.github.io/safuncs/reference/Surv_Power.md)
  : Calculate Power for Survival Experiments
- [`Surv_Pred()`](https://sean4andrew.github.io/safuncs/reference/Surv_Pred.md)
  : Predict Future Survival
- [`Surv_Simul()`](https://sean4andrew.github.io/safuncs/reference/Surv_Simul.md)
  : Simulate Survival Data

## Count Data Analysis

Functions for simulating count data and count power analysis

- [`Con_Simul()`](https://sean4andrew.github.io/safuncs/reference/Con_Simul.md)
  : Simulate Contingency Table
- [`Con_Simul_PR()`](https://sean4andrew.github.io/safuncs/reference/Con_Simul_PR.md)
  : Calculate Positive Rates for Contingency Table

## Pathology Data Analysis

Functions for cleaning, analyzing, and plotting pathology data

- [`Path_Gen()`](https://sean4andrew.github.io/safuncs/reference/Path_Gen.md)
  : Generates Cleaned Pathology Data
- [`Path_Summary()`](https://sean4andrew.github.io/safuncs/reference/Path_Summary.md)
  : Create Summary Statistics from Pathology Data
- [`Path_Table()`](https://sean4andrew.github.io/safuncs/reference/Path_Table.md)
  : Create Flextables from Prevalence Data

## Transform Data

Functions to aid in data transformations

- [`xlsx_trimrow()`](https://sean4andrew.github.io/safuncs/reference/xlsx_trimrow.md)
  : Trim Rows Based on Non-NA Values
