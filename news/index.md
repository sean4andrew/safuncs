# Changelog

## safuncs 2.4.0

*0Jul2025 - Current*

- Revamped function
  [`Surv_Pred()`](https://sean4andrew.github.io/safuncs/reference/Surv_Pred.md).
- Added various arguments to
  [`Surv_Plots()`](https://sean4andrew.github.io/safuncs/reference/Surv_Plots.md).
- Added function
  [`fpr_check()`](https://sean4andrew.github.io/safuncs/reference/fpr_check.md).
- Added various color vectors.
- Added various example datasets.

## safuncs 2.3.0

*18Feb2025* - *01Jul2025*

- Revamped function
  [`Surv_Pred()`](https://sean4andrew.github.io/safuncs/reference/Surv_Pred.md).
- Completed
  [`Surv_Pred()`](https://sean4andrew.github.io/safuncs/reference/Surv_Pred.md)
  documentation.
- Added an argument to
  [`Surv_Gen()`](https://sean4andrew.github.io/safuncs/reference/Surv_Gen.md)
  and
  [`Surv_Plots()`](https://sean4andrew.github.io/safuncs/reference/Surv_Plots.md)
  to account for an additional factor.
- Added data transformation assisting functions
  [`xlsx_row2coln()`](https://sean4andrew.github.io/safuncs/reference/xlsx_row2coln.md)
  and
  [`xlsx_trimrow()`](https://sean4andrew.github.io/safuncs/reference/xlsx_trimrow.md).
- Added miscellaneous functions
  [`silencer()`](https://sean4andrew.github.io/safuncs/reference/silencer.md)
  and
  [`knife()`](https://sean4andrew.github.io/safuncs/reference/knife.md).

## safuncs 2.2.0

*15Dec2024* - *17Feb2025*

- Added function
  [`MultiVar()`](https://sean4andrew.github.io/safuncs/reference/MultiVar.md)
  used to generate a report from multivariate analyses.
- Completed
  [`Surv_Power()`](https://sean4andrew.github.io/safuncs/reference/Surv_Power.md)
  documentation.
- Improved simulation speed for
  [`Surv_Simul()`](https://sean4andrew.github.io/safuncs/reference/Surv_Simul.md).
- Added `multivar_db_ex` as an example multivariate dataset.
- Added `surv_sim_db_ex` as an example simulated survival object from
  [`Surv_Simul()`](https://sean4andrew.github.io/safuncs/reference/Surv_Simul.md).

## safuncs 2.1.0

*14Dec2024* - *15Dec2024*

- Added function
  [`Surv_Power()`](https://sean4andrew.github.io/safuncs/reference/Surv_Power.md)
  used to calculate power for survival experiments.

- Fixed end-TTEs from
  [`Surv_Simul()`](https://sean4andrew.github.io/safuncs/reference/Surv_Simul.md)
  to be integers.
- Changed code in various functions (minor).
